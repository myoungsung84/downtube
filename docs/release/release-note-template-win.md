Architecture: x64
Type: Unpacked portable build

사용 방법:

ZIP 파일 압축 해제
Downtube.exe 실행

⚠ Windows SmartScreen 안내
본 프로그램은 개인 개발자가 직접 빌드하여 배포한 데스크톱 애플리케이션으로,
Microsoft 코드 서명 인증서가 적용되지 않아
Windows SmartScreen 경고가 표시될 수 있습니다.

이는 악성 코드 때문이 아니라,
배포 이력이 많지 않은 신규/개인 배포 프로그램에 대해
Windows에서 기본적으로 표시하는 보안 경고입니다.

실행 방법:

SmartScreen 경고 화면에서 [추가 정보] 클릭
[실행] 버튼 선택

본 프로젝트는 GitHub에서 공개적으로 관리되고 있으며,
빌드 과정과 소스 코드를 직접 확인하실 수 있습니다.

Downtube의 소스 코드는 MIT License로 배포됩니다.
본 애플리케이션은 yt-dlp와 FFmpeg 같은 외부 오픈소스 도구를 사용하며, 각 도구의 라이선스 조건은 해당 프로젝트 정책을 따릅니다.
포함된 외부 도구는 독립적인 실행 파일로 동작합니다.
일부 백신 소프트웨어에서 본 애플리케이션 또는 포함된 외부 도구를 오탐지(false positive)로 분류할 수 있습니다.
